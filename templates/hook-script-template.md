# Hook Script Template

Use these patterns when the source material is strong but the opening is weak.

## Pattern 1: Conclusion First

"The biggest mistake people make when turning long videos into shorts is [mistake]."

## Pattern 2: Curiosity Gap

"This moment worked in the full video, but it would fail as a short unless you changed one thing."

## Pattern 3: Direct Utility

"If you want to get more short clips from one long video, start here."

## Pattern 4: Problem First

"Most long videos do not fail because they lack good moments. They fail because the short starts too late."

## Pattern 5: Contrast

"A clip is not just a shorter video. It is a rebuilt unit with a new opening and a cleaner payoff."

## Pattern 6: Story Turn

"Everything feels normal until the source hits one moment that changes the clip."

## Pattern 7: Audience Filter

"If you are clipping podcasts, this works. If you are doing film recap, it usually does not."

## How To Use These Patterns

1. Pick one payoff
2. Choose the opening pattern that matches the payoff
3. Rewrite the first line for clarity and speed
4. Remove any setup that delays the main point
