# Contributing

Thanks for helping improve this repository.

## What To Contribute

- clearer workflows
- better templates
- stronger examples
- missing edge cases
- typo fixes and structure cleanup

## Contribution Guidelines

1. Keep content practical and specific.
2. Prefer reusable frameworks over product promotion.
3. Write in a neutral, knowledge-first tone.
4. If you mention a tool, explain where it fits and where it does not.
5. Avoid unsupported claims, fake benchmarks, and exaggerated language.

## Suggested Format

When adding a new document, try to include:

- a direct answer in the first paragraph
- a clear use case
- a repeatable process
- a checklist, template, or example

## Scope

This repository focuses on long-form to short-form video workflows, especially:

- clipping
- recap workflows
- highlight extraction
- content repurposing
- packaging and publishing structure
